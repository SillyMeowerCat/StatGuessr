# StatGuessr
## How to Install
1. Make sure you have [tampermoneky](www.tampermonkey.net/index.php) installed. 
2. Install either the [script](https://github.com/SillyMeowerCat/StatGuessr/raw/refs/heads/main/mini.user.js)
3. Enjoy!

## Features
- Profile Cards (displays peak elo, current elo, duels played, worst countries, best countries, and ban status.
- Displaying all players ELOs and ban status while in parties, duels, etc
- Tracking frequency of countries
- Colorcoding ELO

## Whats the difference?
StatGuessr Mini only includes adding profile cards to profiles. Whilst the complete StatGuessr script includes all features.
